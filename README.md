# ExamGuard - Online Exam Management System

## 📌 Project Overview
ExamGuard is a web-based online exam management system built using React (frontend), Node.js + Express (backend), and SQLite database.  
It allows admins to create exams, students to take exams, and view results/leaderboard.

---

## 🚀 Features
- Create and manage exams
- Online exam evaluation system
- Result calculation and storage
- Leaderboard system
- Responsive UI

---

## 🛠️ Tech Stack
- Frontend: ReactJS
- Backend: Node.js, Express.js
- Database: SQLite
- API Testing: Axios
